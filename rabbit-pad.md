Shortcuts (Also work as toggles)
- Ctrl + Enter = Create new file and focus on it
- Ctrl + I = Make the text right before the cursor Italic else switch to italic writing mode if no text is before
- Ctrl + B = Make the text right before the cursor Bold else switch to Bold writing mode if no text is before
- Ctrl + U = Make the text right before the cursor Underline else switch to undeline writing mode if no text is before

- Ctrl + 1 = Heading 1
- Ctrl + 2 = Heading 2
- Ctrl + 3 = Heading 3
- Ctrl + Shift + L = Divider/seperator

Select the text and press ' or " or ( to wrap the selected text.
